# Гайд по Vlkxn для друга

## Что это

Vlkxn — децентрализованная P2P VPN для игр. Позволяет играть по локальной сети (LAN) через интернет, как будто вы в одной комнате.

Мы используем комнату `test` с паролем `12345`.

## Установка (Linux)

### 1. Скачать .deb

```
wget https://github.com/rival-afk/vlkxn/releases/download/v0.3.0/vlkxn_0.3.0_amd64.deb
sudo dpkg -i vlkxn_0.3.0_amd64.deb
```

Если dpkg ругается на зависимости:
```
sudo apt install -f
```

### 2. Запустить

```
sudo vlkxn up --room test --password 12345
```

Терминал должен показать:
- `Connected to room: test`
- `Virtual IP: 10.144.23.x` (адрес будет свой у каждого)
- список интерфейсов, на которых слушает пир

### 3. Проверить подключение

Не закрывая первый терминал, открой второй и проверь:

```
ping 10.144.23.64
```

Если пинг идёт — всё работает, мы в одной виртуальной сети.

> **Важно:** Не закрывайте терминал с Vlkxn — пока он работает, вы в сети. Чтобы остановить: `Ctrl+C`.

## Если не работает

1. **Файрвол** — убедись, что UDP/TCP порты не заблокированы. Vlkxn использует случайные порты (обычно в диапазоне 40000-60000).
2. **Обновление** — скачай последнюю версию с https://github.com/rival-afk/vlkxn/releases
3. **Пинг не идёт, но vlkxn запущен** — подожди 30-60 секунд, пиры ищут друг друга через DHT.
4. **Ошибка "Permission denied"** — нужен `sudo` для создания сетевого интерфейса.
5. **Напиши мне** — я помогу.

## Как остановить

`Ctrl+C` в терминале, где запущен vlkxn. Интерфейс `vlkxn0` исчезнет сам.

## Полезное

- Логи: `RUST_LOG=debug sudo vlkxn up --room test --password 12345`
- Версия: `vlkxn --version`
- Статус: `ip addr show vlkxn0`
