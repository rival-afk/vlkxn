# AGENTS.md — правила работы с Vlkxn

## Язык
- Все commit сообщения писать на **русском**.
- Весь код, комментарии и документация — на **русском** (если явно не указано иное).

## Коммиты
- После каждого изменения в коде (фикс, фича, рефакторинг) делать **git add -A && git commit**.
- Если коммит не прошел из-за хуков или ошибок — исправить и сделать новый коммит.
- **Не** делать amend, force push, пустые коммиты.

## CI/CD
- Релизы собираются **только** через GitHub Actions при пуше тега `v*`.
- Все сборки — на GitHub Actions, VPS/VDS нет.
- Локально собирать только для отладки (cargo build, cargo check).
- Перед тегом проверить: `cargo check --all-targets`.

## Команды
- `cargo check --all-targets` — быстрая проверка компиляции.
- `cargo fmt --all` — перед каждым коммитом.
- Релиз: `git tag vX.Y.Z && git push origin vX.Y.Z`.
